# 照妖镜

> 按 Enter 之前，让 AI 照一下你刚写的 prompt。
> 含混处现查行业数据——不落盘、不编造、不评分。

## 安装

对你 AI 说：

> 帮我装上 github.com/290963249/zhaoyaojing 的照妖镜 skill

## 用法

```
/照 把搜索改快点
```

🪞 S-PERF · "把搜索改快点"
   Meilisearch P99 22ms · 500K 文档基准实测
   Google SRE Book · 必须用 P99 不用平均值

不评分、不审问、不替你写。镜子只反射。

## 11 招

| 招 | 照什么 |
|----|--------|
| S-PERF | "快/高效"没说多快 |
| S-QUANT | "所有/全部"没划边界 |
| S-NFR | "稳定/可靠"没给判据 |
| R-DOD | 没定义什么算做完 |
| G-WHY | 只写了做什么，没写为什么 |
| G-NOGO | 多个目标互冲没取舍 |
| I-SSOT | 同一件事两处说的不一样 |
| I-ADR | 关键决策没记录为什么不选另一个 |
| V-NAME | 一个词你和 AI 理解不同 |
| V-STAKE | 只写了 happy path，没写失败场景 |
| V-LAYER | Why 和 How 混在一起说 |

短补全、闲聊、调 bug 时别用。
