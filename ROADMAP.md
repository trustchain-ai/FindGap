## v0.5 — 11 招照妖（已发布）

**核心范围**
- 穷尽 10 维度 171 条 Gap 根因 → 三维交集（频率×代价×可拦截）→ **精选 11 条**
- **PASS/FAIL 二值门禁**：11 条零命中 = PASS → 可流转下一流程
- **一招定位**：3 条语义检查（S-PERF/S-QUANT/S-NFR）+ 8 条 grep/模板（余下）
- **照出原形句 + 补齐方向**：固定 5 字段 `[FAIL]` 块，给上游迭代
- **瘦身**：SKILL.md 774 → 236 行；manifest.json 砍 60% 字段
- **项目自身作为首个灰度**：dogfood/runs/004 首次全仓扫描

**砍狠清单**（不可逆）：Phase 0 引擎 / G1-G18 67 信号 / severityScheme 五档 / readinessScore 评分 / JSON 第三通道 / 补齐卡片模板 / compatibility 矩阵 / gapIdFormat / 双轨自检契约

**工期**：1 天（2026-06-02）

**退出条件**（4 项，仓内可 grep 验证）：
1. `grep -cE "^\| \*\*[SVIGRP]-" skill/照妖镜.skill.md` ≥ 11
2. `wc -l skill/照妖镜.skill.md` ≤ 300
3. 沉没成本词零命中（`grep -rE "severity|readinessScore|Phase 0" skill/` = 0）
