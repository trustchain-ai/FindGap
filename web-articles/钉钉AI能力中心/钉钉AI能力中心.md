### 1\. 什么是钉钉文档 MCP Server？

钉钉文档 MCP Server 是基于 Model-Context-Protocol（MCP）的智能文档能力组件，支持通过自然语言指令完成以下操作：

-   **查找文档**：按关键词搜索用户有权限访问的文档，或浏览文件夹/知识库下的文件列表
-   **读取文档**：获取文档的 Markdown 内容或元信息（标题、类型、创建时间等）
-   **创建文档与文件夹**：在指定位置创建新文档（支持带初始内容）或文件夹
-   **编辑文档内容**：覆盖或追加 Markdown 内容到文档
-   **精确编辑文档结构**：在 Block（块元素）级别对文档进行插入、更新、删除操作，支持段落、标题、表格、高亮块、列表等多种块类型

适用于知识沉淀、自动摘要、跨应用内容同步、文档自动化生成与编辑等场景。

### 2\. 如何使用钉钉文档 MCP Server？

该服务已在钉钉云端自动部署，无需额外配置。你只需在钉钉对话或 AI 助手中输入自然语言指令，即可调用相关功能。

**支持的典型用法示例：**

-   找一下名为"Q3项目计划"的文档
-   创建一个新文档，标题是"周报模板"，写入本周的工作总结
-   在我的文档里创建一个"项目资料"文件夹
-   帮我把这段内容追加到文档末尾
-   查看这篇文档有哪些章节（列出所有块元素）
-   在文档的第二章前面插入一个新段落
-   把文档中某个段落的内容更新为新的文字
-   删除文档中的第三个块元素
-   浏览"产品知识库"下的所有文件
